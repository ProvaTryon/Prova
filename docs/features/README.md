# Feature Registry

This directory contains consolidated documentation for all completed features in the Clothing Store App.

## Completed Features

### 1. Internationalization (i18n)
- **File**: `internationalization.md`
- **Completed**: October 15, 2025
- **Summary**: Full English/Arabic bilingual support with 820 translation keys (410 per language)
- **Key Files**: `frontend/messages/en.json`, `frontend/messages/ar.json`, all page and component files
- **Related Features**: All UI pages and components

### 2. Dark Mode
- **File**: `dark-mode.md`
- **Completed**: October 15, 2025
- **Summary**: Advanced dark mode with light/dark/system themes, full i18n support, fast transitions (100ms), persistent storage
- **Key Files**: `frontend/components/layout/theme-toggle.tsx`, `frontend/app/layout.tsx`, `frontend/app/globals.css`
- **Related Features**: Navigation, Internationalization

## Documentation Guidelines

1. **When to create a feature document**: After completing any significant feature or major functionality
2. **Document structure**: See `../copilot-instructions.md` Section 0.1 for template
3. **Maintenance**: Update the feature doc when making changes, don't create new docs
4. **Cleanup**: Delete temporary docs after consolidation into feature summaries
